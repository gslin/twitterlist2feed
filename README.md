# twitterlist2feed

Export all media (photos) of a Twitter list into another Atom feed.

Setup
-----

In `~/.config/twitterlist2feed/config.ini`:

    [default]
    consumer_key = x
    consumer_secret = x
    access_token = x
    access_token_secret = x

Usage
-----

    ./twitterlist2feed.pl [screenname] [listname]

Example:

    ./twitterlist2feed.pl gslin fj
